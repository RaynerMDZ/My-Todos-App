# Todo's Web App
In this website, you can create your own to-do list and manage it. You can perform actions such as create a to-do, update a to-do, delete a to-do. 
This website is a single page web application. It is secured with JWT and Spring Security. 

#### Tech Stack:
###### Back-end:
Java 8, Spring Boot 2, Spring Security, JPA/Hibernate, Jackson 2.9.4, Maven, MySQL 8, H2.
###### Front-end:
Angular 7, Bootstrap 4, Restful API, TypeScript, HTML 5, CSS 3, SASS.
